# Lineyou Project
[lineyou](https://github.com/Amazingwujun/lineyou) 基于 `javafx`、`mqtt_3.1.1`，运用 `netty、spring、fxlauncher、jfoenix、fontawesomefx、protobuf` 等技术框架实现的 im 程序，具备基本的聊天交互功能。
